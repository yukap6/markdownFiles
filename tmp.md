URL 和 URI 的区别

* URI（Uniform Resource Identifier）统一资源定位符，包括文件、服务等
* URL (Uniform Resource Location) 资源所处位置

中间的：代理，网关，通道
intermediary：Proxy, Gateway, tunnel

HTTP 建立在 TCP/IP 之上

HTTP 1.0 多数情况下，一个连接只能处理一次请求响应；HTTP 1.1 则有可能一个连接处理多个请求响应。
